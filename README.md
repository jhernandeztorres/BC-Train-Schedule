# Train Schedule
This is an application that uses HTML, CSS, JQuery, and Firebase to create a schedule that changes with time. 


# Starting Out 
1. Clone repo
2. Go to folder in bash/terminal
3. npm install
4. See [notes](#notes) to run the app
5. Enjoy
